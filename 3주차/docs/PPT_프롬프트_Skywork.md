# 3주차 과제 PPT 작성을 위한 Skywork 프롬프트

## 📋 과제 개요
**주제**: AI 기반 데이터 모델링 및 OpenCV를 활용한 결과 시각화  
**목표**: YOLOv8 객체 탐지 모델 학습, 객체 탐지 및 패턴 분석, 결과 시각화  
**데이터셋**: 5개 이미지 (Person, Car, Dog) - OIP (1).jpg ~ OIP (5).jpg  
**기술 스택**: YOLOv8, PyTorch, OpenCV, Matplotlib, Ultralytics
**Git 저장소**: https://github.com/heotaewoong/ai_vision (과제 제출용)

## 📋 과제 요구사항 반영
- **AI 모델 학습 코드 및 실행 결과 문서화**: YOLOv8 모델 학습 과정 및 성능 평가
- **OpenCV를 활용한 결과물 시각화**: 객체 탐지 결과 시각화 및 분석 보고서
- **TensorFlow/PyTorch 기반 모델링**: PyTorch 기반 YOLOv8 모델 활용
- **OpenCV 객체 탐지 및 이미지 분석**: 실시간 객체 탐지 및 바운딩 박스 시각화
- **AI 모델 평가 방법 및 시각화**: 성능 지표 분석 및 종합 대시보드
